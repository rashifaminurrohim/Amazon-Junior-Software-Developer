Project Overview: Build A full-stack shopping list App

Scope

This shopping list app should include:

A front-end interface using HTML, CSS, and JavaScript for adding, modifying, and deleting items.

The app should implement the server-side logic using Spring Boot to handle HTTP requests and manage user sessions.

The app should provide the required RESTful API endpoints that serve the front-end.
